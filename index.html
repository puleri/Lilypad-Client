Goals for Lilypad

                  

  Lilypad is built and maintained by one developer
independent of any company. Some goals for the
application include making it a fully fleshed out
note-taking application comparable to some of the
most popular note-taking applications currently out
there. In its infancy, feature updates for Lilypad
coming soon will include the ability to reset a
forgotten password, an immediate index of notes upon
log-in, and an overall smoother notes interface.
Things Lilypad would love to integrate in future
iterations would include a 'Category' selector with
types such as 'Grocery' or a 'Music Playlist' both of
which would make queries based on your location to find
a grocery store near you, or when a band will preform
next via ticketmaster. For now, please feel free to
give feedback on the issues feed for the github repo.
Happy note taking!
